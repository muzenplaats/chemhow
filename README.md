# kemijn
This is currently a symbolic test for chemical elements, atoms, molecules, compounds, reactions and others.

## Symbolic test
```kemijn
H
H-.
H_2
H-. + H-. <=> H_2                                           [1]
H^+ + H^- <=> H_2                                          [1?]
1^H_2
2^H_2 === D_2
1^H 2^H === HD
2^H 1^H === DH
3^H_2 === T_3
...
He
Li, Li-., Li^+,
Be, Be-., Be^{2+},
B, C, N, O, F, Ne,
...
CH_4, CH_3, CH_3-.,
CH_3-. + .-OH <=> CH_3OH                                    [2]
CH_3^+, -^OH <=> CH_3OH,                                   [2?]
CO_2 + CaO -> CaCO_3                                        [3]
CO_2 === O=C=O === ::O=C=O::
```
